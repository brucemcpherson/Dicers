The Dicers add-on adds functionality to Sheets in some of the same ways as Slicers do for Excel. The unique values in selected columns are shown in 'Dicers'. Selecting one or more of these values will limit the data shown in the sheet to include only the rows that contain these values.

Changes in data are detected automatically, as are movements between sheets.

What are Dicers?
The Dicers add-on provides a simple and visual way of applying filters to data in Google Sheets, and is similar in function and appearance to Slicers in Excel.

Dicer features - standard edition
Any column in a table can become a Dicer. Selecting value(s) will filter the rows in the sheet according to the values selected in each of the dicers.

  • Automatically detects data changes
  • Underlying value types are used for sorting
  • Select single or multiple filter values
  • Resize or rearrange the Dicers
  • Icons ands label have tooltips
  • Tailor behavior and appearance
  • Save settings at document or user level
  • Pause and restart filtering

Pro edition
The Pro edition has all the features of the Standard eidtion, but has additional capabilities, some of which are shown below

  • Find tables automatically on complex pages
  • Filter by background color
  • Save individual Dicer settings across sessions and sheets
  • Apply OR and NOT operations across Dicers
